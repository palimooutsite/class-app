# Class App

Aplikasi ini berisi proyek dasar untuk **Class App**.

## Fitur
- Struktur proyek awal.
- Tempat untuk menambahkan dokumentasi dan panduan penggunaan.

## Cara Menjalankan
1. Pastikan dependensi proyek terpasang.
2. Jalankan perintah sesuai kebutuhan proyek (misalnya `npm install` lalu `npm run dev`).
